CSS

Cascading Style Sheets (CSS) is a style Sheets language used for describing the presentation of a document writthen in a markup language. Although most often used to set the visual style of web pages and user interfaces writthen in HTML and XHTML, the language can be applied to any XML document, including plain XML, svg and XUL, and is applicable to rendering in speech, or on other media.
